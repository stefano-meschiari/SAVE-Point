Kepler's 1st Law:

 

In 1609, Johannes Kepler discovered that when objects interact because of the force of gravity, the smaller object can only travel a few possible paths through space.  This is true for any objects in space, not just planets around their stars!  Any object that feels the pull of gravity, from massive planets like Jupiter to the smallest asteroid, will travel one of these paths.

 

As long as the smaller object has the perfect speed and direction, it can end up in a circular orbit.  In nature, perfectly circular orbits are quite rare!  Based on the small object's distance from the massive central object, there is only one speed that will result in a circular orbit, and it must be traveling perpendicular to direction of gravity's pull.

 

If the small object has a slightly different speed, or if it is travelling in a slightly different direction than it needs for a circular orbit, it will end up following an elliptical (oval) path. The massive object is located at one focus of the ellipse, not the center.

 

 

 

When a small object in space passes by a large object at very high speeds, the small object can continue off into space to be lost forever!  Since the pull of gravity from the massive object still affects the small one, the path of the small object is bent toward the massive object, but not enough to be pulled into orbit.  This ejected path can be a parabola or a hyperbola, depending on exactly how fast the small object is traveling relative to the big object.

 

 

Kepler's amazing discovery was that all of the possible paths (circle, ellipse, parabola, and hyperbola) could be described by the same mathematical model.  Before Kepler discovered this, scientists thought everything in space moved in circular paths!  For more information, look into these shapes, known as Conic Sections.
